# TIB.TJ — MVP: воронка записи на КТ/МРТ

Позиционирование поменялось (v2): вместо справочника врачей/клиник с прямыми контактами — одна воронка. Пациент отвечает на несколько вопросов (какое исследование, какая область, есть ли направление) и попадает в WhatsApp уже с готовым осмысленным сообщением. Ты вручную распределяешь пациентов по клиникам — это и есть рычаг для переговоров с клиниками.

Дизайн (v3): светлый "стеклянный" интерфейс — полупрозрачные карточки с backdrop-blur поверх мягкого бело-мятного фона, крупные скругления, бирюзовый + тёплый золотой акценты. Переключатель RU/TJ, адаптив под мобильные с sticky-кнопкой записи внизу экрана.

## Версия 9 — чистая пересборка, залей с нуля

Из-за старых файлов, которые копились в репозитории с первых версий (справочник врачей/клиник, конфиги Tailwind), несколько раз ловили ошибки сборки на Vercel. Чтобы закрыть это раз и навсегда, проще всего полностью очистить репозиторий и залить всё заново из этого архива, а не удалять файлы по одному.

**Как это сделать:**
1. Зайди в репозиторий на GitHub.
2. Выдели все файлы и папки на главной странице репозитория (или зайди в каждую папку и удали её содержимое — значок корзины на файле → Commit changes). Проще всего: удали каждый файл/папку из списка на главной странице репозитория по одному через корзину.
3. Когда репозиторий станет пустым — через **Add file → Upload files** перетащи туда всё содержимое папки из этого архива (v9) целиком, включая скрытые файлы `.gitignore` и `.env.example`.
4. **Commit changes**.

В этом архиве уже нет ничего лишнего — ни справочника врачей/клиник, ни Tailwind. Дальше пересобирать репозиторий с нуля не понадобится, только заменять отдельные файлы через тот же **Add file → Upload files**, когда я буду присылать точечные правки.

## Что где лежит
- `app/page.js` — вся посадочная страница (hero, шаги, сравнение, КТ/МРТ карточки, анкета записи, FAQ)
- `app/about`, `app/oferta`, `app/privacy`, `app/consent` — страницы-заглушки, текст правится прямо в этих файлах
- `app/uslugi/[slug]/page.js` + `lib/seoPages.js` — 20 SEO-страниц, редактируются только через `lib/seoPages.js`
- `components/BookingForm.js` — анкета перед WhatsApp (исследование → область → направление → время → готовое сообщение → согласие на обработку данных)
- `components/Header.js`, `Footer.js`, `StickyCta.js` — шапка, подвал, мобильная плашка снизу
- `lib/whatsapp.js` — номер WhatsApp/телефона в одном месте (`992788880303`) — поменяй здесь, если номер изменится
- `lib/examinations.js` — список областей обследования для КТ и для МРТ (отдельные списки)
- `lib/i18n.js` — все тексты на русском и таджикском, включая текст согласия и заглушки для новых страниц
- `app/globals.css` — вся стилистика и анимации
- `supabase/schema.sql` — схема БД под клиники/врачей, пригодится на следующем этапе, когда будешь официально подключать партнёров

## Деплой обновлений
Vercel сам передеплоит сайт при новом коммите в GitHub — ничего дополнительно нажимать не нужно, только проверить вкладку **Deployments**, что сборка прошла успешно (зелёный статус Ready).
